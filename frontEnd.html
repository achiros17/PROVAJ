Oggi 14 Maggio abbiamo iniziato a esplorare il linguaggio JavaScript, uno degli elementi fondamentali per lo sviluppo web interattivo. In questa giornata abbiamo visto le basi per dichiarare variabili e come utilizzare console.log per stampare messaggi.

JavaScript è un linguaggio di programmazione che permette di aggiungere interattività alle pagine web. Abbiamo visto le differenze tra var, let, e const per dichiarare variabili e come stampare messaggi nella console utilizzando console.log().

Un esempio di codice per iniziare a lavorare con le variabili in JavaScript potrebbe essere il seguente: 

let nome = "Alessio";
let eta = 25;
const professione = "Sviluppatore Web";

console.log("Ciao, mi chiamo " + nome + ", ho " + eta + " anni e sono un " + professione);

Registrare un evento

Oggi 15 Maggio abbiamo selezionato gli elementi del DOM in Javascript.
Gli elementi del DOM possono essere selezionati in vari modi :

---Stampa elemento del Dom per id
  let paragraph=document.getElementById("pr");
  console.log(paragraph);

---Stampa elementi del Dom per nome del tag
  let par =document.getElementsByTagName("p");
  console.log(par);


---Stampa elementi del Dom per nome della classe
  let blocchi=document.getElementsByClassName("day-section");
console.log(blocchi);


---Con un ciclo for modifico il colore di sfondo di tutti gli elementi con la classe "day-section"
  for(let i=0;i (<) blocchi.length;i++){
  blocchi[i].style.backgroundColor="bisque"
}

Selezionando gli elementi che hanno come nome della classe "day-section" possiamo modificare lo stile, in questo caso lo sfondo attraverso il ciclo for che ciclerà gli elementi fino a quando l'indice (i) sarà minore della lunghezza di tutti gli elementi

Quinto Giorno

Regole CSS

Oggi 16 Maggio abbiamo approfondito regole di stile CSS con il selettore :root che rappresenta l'elemento radice di un documento HTML.Viene spesso usato per definire variabili CSS globali che poi possono essere riutilizzate in tutto il foglio di stile.

:root {
  --main-color: #3498db;
  --font-size: 16px;
}

Create queste variabili possiamo riutilizzarle e applicarle ad altri elementi con la sintassi var(--nomeVariabile); tipo : 

body {
  color: var(--main-color);
  font-size: var(--font-size);
}


Importante e ormai fondamentale è il Response Design. Infatti prima dell'avvento dei nuovi dispositivi mobile come tablet e smartphone che hanno una dimensione dello schermo minore rispetto al pc,è stato necessario rendere versatile quindi il codice anche per questi dispositivi. Da sempre utilizzati quindi per le dimensioni di un'immagine o di un testo erano i Pixel,un'unità di misura fissa che quindi non riesce ad essere responsive.

Con l'avvento dei nuovi dispositivi è stato necessario utilizzare unità di misura relative come i rem e i em.
REM (Root Element) è un'unità di misura relativa che si basa sulla dimensione del font radice (root font size) del documento. In altre parole, 1rem è uguale alla dimensione del font definita nell'elemento :root o nell'elemento html. Ad esempio, se la dimensione del font radice è 16px, 1rem sarà uguale a 16px. 

:root{
  font-size: 16px;
}

p {
  font-size: 1.5rem; 
}
In questo esempio vediamo rem applicato all'elemento paragrafo, farà riferimento alla dimensione del font radice e NON al suo genitore diretto che potrebbe essere un div 

La dimensione del font del paragrafo sarà quindi 1.5 volte la dimensione del font radice, quindi 1.5 * 16px = 24px.

EM (Element) è un'unità di misura relativa che si basa sulla dimensione del font dell'elemento genitore diretto. In altre parole, 1em è uguale alla dimensione del font dell'elemento in cui viene utilizzato. Ad esempio, se un elemento ha una dimensione del font di 20px e all'interno di esso si utilizza 1em, il valore sarà 20px. 
:root {
  font-size: 16px;
}

div {
  font-size: 20px;
}

p {
  font-size: 1.5em; 
}
In questo esempio vediamo em applicato all'elemento paragrafo,farà riferimento però alla dimensione del font del suo genitore diretto quindi il div e NON alla radice 

La dimensione del font del paragrafo sarà quindi 1.5 volte la dimensione del font del div padre, quindi 1.5*20px=30px

BOOTSTRAP

Oggi 19 Maggio dopo aver visto le basi di HTML, CSS e JavaScript, abbiamo iniziato a esplorare Bootstrap, un framework CSS che semplifica la creazione di layout reattivi e design accattivanti. Bootstrap offre una serie di classi predefinite che possiamo utilizzare per stilizzare i nostri elementi HTML in modo rapido ed efficiente.

Bootstrap è un framework front-end open source per lo sviluppo di siti e applicazioni web. È stato creato da Twitter ed è progettato per semplificare il processo di creazione di layout reattivi e design accattivanti.
